# Required Files by Domain

Before acting on a domain, load these files first:

| Domain | Required Files |
|--------|---------------|
| **Financial** | `data/brain/topics/Finance.md`, finance tasks from `state/tasks.yaml`, anti-001, anti-002, learn-010, learn-045 |
| **Meeting prep** | `data/brain/people/{person}.md`, calendar event, `state/hot-context.md`, learn-041 |
| **Career/consulting** | `data/brain/projects/{project}.md`, relevant emails (`gog gmail search`), `state/hot-context.md` Active Threads |
| **Content** | `data/brain/content/voice-dna.md`, `data/brain/content/voice-profile.md`, `data/brain/articles/` relevant drafts, learn-021, learn-043 |
| **Task update** | `state/tasks.yaml` (read the actual task), email/calendar for latest signal, then update |
| **Body/Health** | `state/body.yaml` summary section, recent workout logs, medical clearance, biometrics |
| **Infrastructure** | `state/system-overview.yaml`, `TOOLS.md`, `HEARTBEAT.md`, `agents/registry.yaml`, current OpenClaw version |
| **Startup (Digital Twin Co)** | `data/brain/projects/Digital Twin Co.md`, Anna/Gal person files, learn-028, learn-033 |

When spawning sub-agents, include the relevant required files in the task prompt — don't assume the sub-agent will find them.
